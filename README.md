## Requirements

- Install zappajs-partials

```code
npm install zappajs-partials
```
- Install zappajs

```code
npm install zappajs
```
- Install supervisor

```code
npm install supervisor -g
```

- Install Jasminejs

```code
npm install jasmine-node -g
```

## Running

- App:

```code
supervisor run.coffee
```
- Tests

```code
jasmine-node --coffee spec/
```
